# 3D 우주 포트폴리오 개발 계획

## 프로젝트 개요
우주를 배경으로 무중력 상태의 페이지들이 떠있는 3D 포트폴리오 웹사이트

## 핵심 기능
- 우주 배경 (별, 성운 효과)
- 3D 공간에서 떠다니는 포트폴리오 카드들
- 마우스/터치로 3D 공간 회전
- 카드 선택 시 확대 및 상세 내용 표시
- 부드러운 애니메이션 전환

## 기술 스택
- Three.js (CDN) - 3D 렌더링
- HTML5 / CSS3 / Vanilla JavaScript
- CSS 애니메이션 및 트랜지션

## 파일 구조 (최소화)
```
Portfolio/
├── index.html    # 메인 HTML
├── style.css     # 스타일시트
├── main.js       # 3D 로직 및 인터랙션
├── plan.md       # 개발 계획
└── CLAUDE.md     # AI 가이드라인
```
